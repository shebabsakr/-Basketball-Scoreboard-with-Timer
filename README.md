A simple basketball scoreboard web app built using HTML, CSS, and JavaScript.
This project allows users to:
✅ Increment home and guest scores (+1, +2, +3)
✅ Reset scores
✅ Set a custom countdown timer
✅ Start and reset the timer
✅ Display dynamic font size when time is under 10 seconds
🚀 Features
🏠 Home & Guest Score Controls
Add 1, 2, or 3 points
Reset both scores to 0
⏱ Timer Functionality
Enter custom time in seconds
Countdown updates every second
Displays "Time Finished!!!" when complete
Larger font when under 10 seconds remaining
🛠 Technologies Used
HTML5
CSS3
Vanilla JavaScript (No libraries or frameworks)
